Welcome to ROUGE Metric!
========================

The rouge-metric package provides a fast Python implementation of full `ROUGE <https://www.aclweb.org/anthology/W04-1013/>`_ metrics and a Python wrapper of the official ``ROUGE-1.5.5.pl`` script.

The project is hosted on GitHub `<https://github.com/li-plus/rouge-metric>`_
